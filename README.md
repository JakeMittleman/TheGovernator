# TheGovernator
## To Run Yourself
Create a file *twitterCreds.py* and fill in these API keys:
- ACCESS_TOKEN
- ACCESS_TOKEN_SECRET
- CONSUMER_KEY
- CONSUMER_SECRET

On line 65, put in the username of the user you would like this effect done to. Additionally, method *reformat_text(self, text)* can be changed to have any effect.

**NOTE:** These need to be from Twitter's v1.1 API
